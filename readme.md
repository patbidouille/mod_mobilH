Pour commencer, utilisez :

- main_test_diag : qui testera la connexion de la carte SIM.
- main_test_sms : pour tester les SMS sur votre portable
- main_test_mqtt : envoi un message de test sur votre broker (HA)
- main_test_capteur : qui test les capteurs I2C
- main : programme principal
